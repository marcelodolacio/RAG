# RAG - (Retrieval-Augmented Generation)

Essa aplicacao é uma aplicacao pra treinar o conceito de vectors e embeddings onde foi feito um RAG com a documentacao de 2019 do futebol americano. Inspirado em um módulo da pós graduação em engenharia de software com IA aplicada.

Tecnologias utilizadas:

- Neo4j
- OpenAI
- Node.js
- HTML
- CSS
- JavaScript
