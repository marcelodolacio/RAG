# RAG (Retrieval Augmented Generation)

Essa aplicacao é uma aplicacao pra treinar o conceito de RAG, utilizei da CBF de 2026 com as regras do futebol. Inspirado em um módulo da pós graduação em engenharia de software com IA aplicada.

Tecnologias utilizadas:

- Neo4j
- OpenAI
- Node.js
- HTML
- CSS
- JavaScript
- LangChain
- OpenRouter
